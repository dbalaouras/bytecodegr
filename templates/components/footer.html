About this Webpage

This is my personal web page and was built within very few days.Thanks to the wonderful technologies
other people have built, you will be surprised how fast you can create personal websites!
These are the technologies I used:

  • Twitter's Bootstrap framework
  • A theme by uexel.us
  • Flask web microframework for python
  • JQuery library for JavaScript
  • Jinja template engine
  • Apache HTTP Server
  • Modwsgi module for Apache

The rest was some pretty basic python programming, some JavaScript/JQuery and some CSS3.
Hope you like it! But even if you don't, please go ahead and send me your note...I'd appreciate any feedback!
